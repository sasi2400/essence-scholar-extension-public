API Keys

API keys are required for all models (Gemini, GPT, and Claude). You'll need to provide the appropriate API key for your selected model.

üí° How to get a Google AI API key (Optional but recommended):

1. Go to Google AI Studio
2. Sign in with your Google account
3. Click "Create API Key"
4. Copy the key and paste it above

üí° How to get an OpenAI API key (Required for GPT models):

1. Go to OpenAI Platform
2. Sign in or create an account
3. Navigate to API Keys section
4. Create a new secret key

üí° How to get a Claude API key (Required for Claude models):

1. Go to Anthropic Console
2. Sign in or create an account
3. Navigate to API Keys section
4. Create a new key
